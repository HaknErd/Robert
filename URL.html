URL and Domains

A Uniform Resource Locator (URL), which is also known as a web address is an identifier to locate certain resources on the internet

A Domain Name Server (DNS) is a server that matches website hostname to the matching IP address

But a Domain Name Server (DNS) and Uniform Resource Locator (URL) are not the same.

For example if you type in a certain web address (URL) (like Youtube) into your browser the browser needs to know where to find the web address (like a real address).

This is done via a DNS lookup
